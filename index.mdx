## Get started in minutes

<Steps>
  <Step title="Download the Compose file">
    Pull the production Docker Compose file from the repository.

    ```bash
    curl -OL https://raw.githubusercontent.com/ZimengXiong/ExcaliDash/main/docker-compose.prod.yml
    ```
  </Step>
  <Step title="Start ExcaliDash">
    Pull images and start the containers.

    ```bash
    docker compose -f docker-compose.prod.yml pull
    docker compose -f docker-compose.prod.yml up -d
    ```
  </Step>
  <Step title="Open the dashboard">
    Navigate to [http://localhost:6767](http://localhost:6767) and start drawing.
  </Step>
</Steps>

<Note>
  ExcaliDash is in BETA. Back up your data regularly and review the [deployment hardening guide](/deployment/docker) before exposing it to production traffic.
</Note>
